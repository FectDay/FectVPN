# FectVPN
The official FectDay Telegram bot for getting the FectVPN beta version.

# Compatible Apps
Our configuration files and keys only work with applications/clients from Amnesia, such as AmneziaVPN and AmneziaWG. DefaultVPN is also suitable for iOS.

**Windows** — AmneziaWG, AmneziaVPN
**Android** — AmneziaWG, AmneziaVPN
**iOS** — AmneziaWG, AmneziaVPN, DefaultVPN
